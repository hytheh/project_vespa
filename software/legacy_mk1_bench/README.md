# legacy_mk1_bench — ARCHIVED

Firmware for the **mk.1 stepper test bench**, kept for reference only.

It is **not** part of the current three-node architecture and is **superseded by
[`../motion_node`](../motion_node)** (STM32G431 + BLDC/FOC + FDCAN). The doxygen
headers below read like maintained code — they are not. Nothing here is built or
flashed by the current system; do not treat it as the active motion firmware.
